# AI Interview Question Generator
A web application that generates random interview questions and answers for Java, DBMS, Operating Systems, and Computer Networks.
AI Interview Question Generator is a web application that helps students prepare for technical interviews by generating random questions and answers from Java, DBMS, Operating Systems, and Computer Networks.

## Features

- Random Interview Questions
- Detailed Answers
- Difficulty Levels
- Multiple Subjects
- Responsive Design

## Technologies Used

- HTML
- CSS
- JavaScript
